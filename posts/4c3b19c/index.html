OnePlus8/8Pro恢复模式(Recovery)

一加8/8P官方REC

氧OS的官方REC：8P[Recovery][Stock OOS]Oxygen OS Stock recovery with ADB enabled | XDA Developers Forums。

由于我们使用的是氢OS，这个REC是一直存在的，不用担心会被刷掉。假设它被刷掉，重新刷个全量包官方REC就回来了。因此，你压根不需要担心这个REC被刷掉。

一加8/8P无法永久刷入Recovery

两种刷入Rec方式

刷入Recovery的方式有两种，flash和boot。

To boot:

目前只支持boot刷入Rec

但是现在一加8/8P只能使用boot方式，这点XDA-mauronofrio在他发布的TWRP帖子中有提及：

Acutally seems oneplus (or directly android 10) re-introduced the recovery partition so you should only flash it in recovery partition or you can try to boot it from fastboot. For now only boot it.

…

DON’T FLASH THE TWRP, please only boot it for now

使用fastboot flash命令后手机会提示"Qualcomm … crashdump"，强制关机重进Recovery后发现变回官方REC。

推荐使用fastboot boot，这种方式相当于刷入临时TWRP。什么意思？一加8/8P使用fastboot boot刷入TWRP后，只是相当于把TWRP进行加载，在下次引导恢复模式时，还是会加载回官网的Rec。
